- Exploration of vision language model that performs image segmentation (Med-VLM) using a language description of the organ's anatomical location. Applied method to a smaller CT organ dataset and performed segmentation on 2D CT slices, compared to the original implementation that used multiple large datasets and segmented using the full 3D CT images
